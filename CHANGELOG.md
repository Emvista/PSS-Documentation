## Notes pour l'anglais

La langue est passée en paramètre. Par défaut c'est le français. Les langues disponibles sont : français (fr) et anglais (en).   

```JSON
{
	"text": "TEXT",
	"parameters": [
		{
			"name": "lang",
			"value": "LANG"
		}
	]
}
```
avec LANG : fr (par défaut) ou en.
